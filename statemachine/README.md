# Experimental State Machine Prototype

This directory contains a standalone experimental state-machine prototype that predates the current Pipeline Services release surface.

It is **not** currently part of:
- the main Pipeline Services build
- the public `v0.1.0` API guarantee
- the portability-contract compatibility promise

The code is kept for reference and exploration, but launch-facing documentation and compatibility guarantees should be read from the main repo root, `docs/PROJECT_STATUS.md`, and the active language ports.
