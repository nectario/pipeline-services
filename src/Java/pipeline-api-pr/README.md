# `pipeline-api-pr`

This directory is an incubating Java API work area for Pipeline Services.

It is intentionally kept in the repository for design exploration, but it is **not**:
- part of the main Maven release build
- part of the public `v0.1.0` compatibility surface
- a supported API promise for the upcoming public launch

Treat the contents here as exploratory material. The public Java release surface for `v0.1.0` is the main `src/Java/pipeline-api/` module plus the modules included by the root Maven build.
